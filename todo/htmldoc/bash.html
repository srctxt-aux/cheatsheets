grp ## Grouping Commands

This doesn't create a new process.

Grouping commands for { CMDS; } 

{ 
  CMDS
 }

convenience:

{ 
    echo "Hello"
    echo "-----"
    echo " bye"
} > myfile.txt

or to pipe stuff: { echo "Hello" && echo "-----" && echo " bye"; } | while read a ; do echo "x $a"; done

set ## The set command

Read commands without executing them. This command is useful for syntax checking. Exit immediately if any simple command gives an error. Print each line of script as it is executed. Display each command expansion as it is executed.
